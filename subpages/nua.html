Nua - Programming Language

Nua is a work-in-progress lightweight scripting language, designed to be used in games for creating a moddable user interface. The sandboxed nature of the language allows developers to give power-users of their game the ability to modify the controls of the game without any possibility of using it to cheat, while the simplicity of the language makes it easy for developers to implement.

An online bytecode compiler can be found at https://galaxyshard.github.io/nua/compiler.html. The interpreter is currently unavailable
